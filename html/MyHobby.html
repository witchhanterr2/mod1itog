Первый сайт, который до сих пор работает, был создан в 1991 году и его можно посетить по адресу: http://info. cern. ch/hypertext/WWW/TheProject. html. Ада Лавлейс считается первым программистом. В 1843 году она работала вместе с Чарльзом Бэббиджем, изобретателем механических компьютеров, над созданием первой компьютерной программы для одной из его машин — аналитической машины. Язык JAVA ранее назывался Oak. Он был назван в честь дуба, выросшего за пределами офиса Джеймса Гослинга (Гослинг наиболее известен как основатель и ведущий разработчик языка программирования Java). Позже его переименовали в Ява, от Java coffee (разновидность кофе из Индонезии). Во время создания языка программирования Java проект Green Project компании Sun Microsystem разработал свою первую операционную демонстрацию — интерактивный портативный контроллер для домашних развлечений, известный как Star 7. -экранный пользовательский интерфейс. Он был разработан одним из художников-графиков — Джо Палрангом. Позже он работал над многими популярными анимационными фильмами, такими как «Шрек», «За изгородью» и многими другими.
